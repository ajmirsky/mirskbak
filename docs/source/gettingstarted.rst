Installation
---------------------------------


install using pip:
 
``pip install git+ssh://git@github.com/ajmirsky/mirskbak.git``
 
or manually:

   git clone git@github.com:ajmirsky/mirskbak.git
   cd mirskbak
   python setup.py install
